# Data Transformation and Feature Creation

## Project Overview

This project demonstrates data transformation and feature creation using Python and Pandas. A sales dataset is used to perform transformations, create derived features, and generate meaningful business insights.

## Objectives

* Load and analyze a dataset
* Perform data transformation operations
* Create new features from existing columns
* Apply map(), apply(), and applymap() methods
* Generate business insights through analysis and visualization

## Dataset Columns

* OrderID
* CustomerName
* Product
* Category
* Quantity
* Price
* Discount
* Region

## Features Created

* TotalSales
* DiscountAmount
* FinalAmount
* Profit
* RegionCode
* RevenueCategory

## Methods Used

* map()
* apply()
* applymap()
* groupby()
* Data visualization using Matplotlib

## Analysis Performed

* Category-wise sales analysis
* Region-wise sales analysis
* Revenue category distribution
* Product performance analysis

## Tools and Libraries

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Conclusion

The project successfully demonstrates data transformation and feature creation techniques. New features were derived from existing data, and insights were obtained through analysis and visualization.
